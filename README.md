# Event Lead Intelligence Platform

## Overview

A lead intelligence platform built using Java Spring Boot, PostgreSQL, and React to automate event-based prospect discovery and management.

## Problem Statement

Sales teams spend significant time manually searching for exhibitors, attendees, companies, and decision-makers across industry events.

This platform streamlines lead discovery, management, filtering, and outreach preparation.

## Features

- Lead Management
- Event Tracking
- Search & Filtering
- Lead Scoring
- Excel/CSV Export
- AI-Assisted Lead Classification
- Dashboard Analytics
